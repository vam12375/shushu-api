# Third-Party Licenses

本文件用于汇总项目所依赖的第三方开源组件许可证信息。

> 说明：此为占位文件，供 Docker 构建阶段归档使用（镜像内 `/licenses/`）。
> 完整的第三方依赖许可证以各依赖自身仓库及 `go.mod` / `web/default/package.json` 中声明为准。

本项目主要依赖的开源组件包括但不限于：

- Go 标准库及 Go 生态依赖（详见 `go.mod`）
- Gin、GORM 等后端框架
- React 19、Rsbuild、Base UI、Tailwind CSS 等前端框架（详见 `web/default/package.json`）

各组件的许可证条款归属其各自的版权所有者。
