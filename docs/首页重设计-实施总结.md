## 🛠️ 技术实现细节

### 零新依赖 ✅

所有功能基于现有技术栈实现，符合 YAGNI 原则：

- **动画**: `tw-animate-css` + `landing-animate-*` 类
- **图标**: `lucide-react` (已安装)
- **颜色**: Tailwind 默认色板
- **布局**: Tailwind Grid + Flexbox

### 关键技术点

1. **Neo-Brutalism 硬阴影**：
   ```tsx
   shadow-[4px_4px_0_rgb(8,145,178)]
   hover:shadow-[2px_2px_0_rgb(8,145,178)]
   ```

2. **SVG 路径动画**（HowItWorks 箭头）：
   ```tsx
   <animate
     attributeName="stroke-dashoffset"
     from="0"
     to="24"
     dur="1s"
     repeatCount="indefinite"
   />
   ```

3. **CSS 变换组合**：
   ```tsx
   hover:scale-105 hover:rotate-1
   transition-all duration-300
   ```

4. **Memphis 风格装饰**：
   - 纯色几何图形 + `blur-3xl` 柔化
   - 分层叠加不同透明度

## 🐛 已知问题与解决方案

### 问题 1: 外部图标加载失败

**现象**: CC Switch 图标因网络原因加载失败

**解决方案**: 已添加 fallback 机制，显示 "CC" 文字徽章

**代码位置**: `hero.tsx:208-220`

---

### 问题 2: 某些浏览器不支持 CSS `backdrop-filter`

**现象**: CTA 区域按钮毛玻璃效果在旧浏览器不显示

**影响**: 视觉效果轻微降级，不影响功能

**解决方案**: 已添加 `bg-white/10` 作为 fallback
